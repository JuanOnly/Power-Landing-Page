instalale un paquete llamado npm-check-updates de forma global, utilizando la terminal:
   npm install -g npm-check-updates

 cada vez que requiera conocer el estado de las dependencias, ejecute el comando nc. 
 Si existen dependencias por actualizar se le solicitará que ejecute el comando    
 ncu -u y por último que ejecute npm install.

npm install -g win-node-env
npm install dayjs

 To go live and check website-  npm run dev

 Si ejecuta desde la consola npm run build el sistema generará los cambios finales para la puesta en producción de la aplicación. 
 Por esta razón sólo se recomienda usar este comando cuando vaya a generar la versión final, mientras esto no ocurra, 
 use  npm run dev para entrar en modo desarrollador y monitorear cada cambio.
